# Calculadora Binária (CLI only)
## Projeto realizado durante graduação no ensino superior técnologo em Gestão de T.I

#### Bases utilizadas:

- Binário
- Decimal
- Hexadecimal
- Octal
  
#### Tipos de cálculos disponíveis:

- Conversões de bases
- Adição
- Subtração
- Multiplicação
- Divisão
